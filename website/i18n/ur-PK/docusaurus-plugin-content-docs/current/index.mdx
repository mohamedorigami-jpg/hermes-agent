## انسٹال کریں

### Windows یا macOS

کمانڈ لائن اور ڈیسک ٹاپ ایپلیکیشنز کو آسانی سے انسٹال کرنے کے لیے، ہماری ویب سائٹ سے [Hermes Desktop انسٹالر ڈاؤن لوڈ کریں](https://hermes-agent.nousresearch.com/desktop) اور اسے چلائیں۔

### Hermes Desktop کے بغیر:

Hermes Desktop کے بغیر صرف کمانڈ لائن انسٹالیشن کے لیے، درج ذیل کمانڈ چلائیں:

#### Linux / macOS / WSL2 / Android (Termux)

```bash
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
```

#### Windows (مقامی)

پاورشیل میں چلائیں:

```powershell
iex (irm https://hermes-agent.nousresearch.com/install.ps1)
```

مکمل **[انسٹالیشن گائیڈ](/getting-started/installation)** دیکھیں کہ انسٹالر کیا کرتا ہے، فی صارف بمقابلہ روٹ لے آؤٹ، اور Windows کے مخصوص نوٹ۔

:::tip کام کرنے والی ایجنٹ کا تیز ترین راستہ
انسٹال کرنے کے بعد، `hermes setup --portal` چلائیں — ایک OAuth میں ایک ماڈل اور تمام چار Tool Gateway ٹولز (ویب سرچ، امیج جنریشن، TTS، براؤزر) شامل ہیں۔ [Nous Portal](/integrations/nous-portal) دیکھیں۔
:::

## Hermes Agent کیا ہے؟

یہ کوئی کوڈنگ کاپائلٹ نہیں جو IDE سے جڑا ہو یا کسی ایک API کے گرد لپٹا ہوا چیٹ بوٹ ہو۔ یہ ایک **خود مختار ایجنٹ** ہے جو جتنا زیادہ چلتا ہے اتنا ہی زیادہ قابل ہوتا جاتا ہے۔ یہ جہاں آپ رکھیں وہاں رہتا ہے — ایک $5 VPS، GPU کلسٹر، یا سرور لیس انفراسٹرکچر (Daytona, Modal) جو بیکار ہونے پر تقریباً کچھ خرچ نہیں کرتا۔ اس سے Telegram پر بات کریں جب یہ کلاؤڈ VM پر کام کر رہا ہو جس میں آپ خود کبھی SSH نہیں کرتے۔ یہ آپ کے لیپ ٹاپ سے منسلک نہیں ہے۔

## اہم فیچرز

- **بند سیکھنے کا لوپ** — ایجنٹ کی زیر نگرانی میموری جس میں وقتاً فوقتاً یاد دہانیاں، خود مختار مہارت کی تخلیق، استعمال کے دوران مہارت کی خود بہتری، FTS5 کراس سیشن یادداشت جس میں LLM خلاصہ کاری، اور [Honcho](https://github.com/plastic-labs/honcho) جدلیاتی صارف ماڈلنگ شامل ہے
- **کہیں بھی چلتا ہے، صرف آپ کے لیپ ٹاپ پر نہیں** — 6 ٹرمینل بیک اینڈز: لوکل، Docker، SSH، Daytona، Singularity، Modal۔ Daytona اور Modal سرور لیس مستقل پیش کش کرتے ہیں — بیکار ہونے پر آپ کا ماحول ہائبرنیٹ ہو جاتا ہے، جس کی قیمت تقریباً کچھ نہیں ہوتی
- **جہاں آپ ہیں وہاں رہتا ہے** — CLI، Telegram، Discord، Slack، WhatsApp، Signal، Matrix، Mattermost، Email، SMS، DingTalk، Feishu، WeCom، Weixin، QQ Bot، Yuanbao، BlueBubbles، Home Assistant، Microsoft Teams، Google Chat، اور مزید — ایک گیٹ وے سے 20+ پلیٹ فارمز
- **ماڈل ٹرینرز کے ذریعے بنایا گیا** — [Nous Research](https://nousresearch.com) کی تخلیق، وہی لیب جس نے Hermes، Nomos، اور Psyche بنائے۔ [Nous Portal](https://portal.nousresearch.com)، [OpenRouter](https://openrouter.ai)، OpenAI، یا کسی بھی اینڈ پوائنٹ کے ساتھ کام کرتا ہے
- **شیڈولڈ آٹومیشنز** — کسی بھی پلیٹ فارم پر ڈیلیوری کے ساتھ بلٹ ان کرون
- **منتقل اور متوازی کرتا ہے** — متوازی کام کے بہاؤ کے لیے الگ تھلگ ذیلی ایجنٹ بنائیں۔ `execute_code` کے ذریعے پروگرامیٹک ٹول کالنگ کثیر مرحلہ پائپ لائنوں کو ایک ہی انفرنس کال میں تبدیل کرتی ہے
- **اوپن سٹینڈرڈ مہارتیں** — [agentskills.io](https://agentskills.io) کے ساتھ مطابقت رکھتی ہیں۔ مہارتیں قابلِ نقل، قابلِ اشتراک، اور Skills Hub کے ذریعے کمیونٹی کے تعاون سے تیار کردہ ہیں
- **مکمل ویب کنٹرول** — تلاش، نکالنا، براؤزنگ، وژن، امیج جنریشن، TTS — ایک سبسکرپشن [Nous Portal](/integrations/nous-portal) کے ذریعے یہ سب اکٹھے فراہم کرتا ہے
- **MCP سپورٹ** — توسیعی ٹول صلاحیتوں کے لیے کسی بھی MCP سرور سے جڑیں
- **تحقیق کے لیے تیار** — بیچ پروسیسنگ، ٹریجیکٹری ایکسپورٹ، Atropos کے ساتھ RL ٹریننگ۔ [Nous Research](https://nousresearch.com) کی تخلیق — وہی لیب جس نے Hermes، Nomos، اور Psyche ماڈلز بنائے
